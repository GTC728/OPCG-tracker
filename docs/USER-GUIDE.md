# OPCG Tracker User Guide

## 這個網站做什麼

OPCG Tracker 幫你記錄 OPCG 對局、管理玩家和 Leader、查看統計、匯入舊 Excel、匯出長期備份，並支援手機和電腦共用資料。

## 第一次使用

1. 打開網站。
2. 到 `設定`。
3. 進入 `玩家管理`。
4. 新增你常用的玩家。
5. 回到 `記錄`。
6. 建立新對局並選擇玩家和牌組。
7. 對局完成後按勝方。

## 新增玩家

1. 到 `設定 → 玩家管理`。
2. 按 `新增`。
3. 輸入玩家名稱。
4. 如有常用暱稱，可填入別名。
5. 儲存後，新對局就可以選到這位玩家。

## 記錄對局

1. 到 `記錄`。
2. 按 `＋ 新對局`。
3. 選玩家 A、玩家 B。
4. 搜尋 Leader 或牌組。可以輸入 `OP01`、`ST10`、`EB01`、leader 名稱或本地別名。
5. 如果知道先攻，可以記錄先攻。
6. 對局完成後，直接按勝方。

## 查看統計

到 `統計` 查看：

- 總對局數。
- 先攻勝率。
- 玩家表現。
- 牌組表現。
- matchup 對位。
- 玩家或牌組 profile。

## 修改歷史

到 `歷史` 可以：

- 篩選玩家、牌組、日期。
- 編輯完成對局。
- 複製舊對局重新開局。
- 軟刪除錯誤對局。
- 還原已刪除對局。

## 匯入舊 Excel

1. 到 `設定 → 匯入 / 匯出`。
2. 按 `選擇 Excel / CSV 檔案`。
3. 選擇 `.xlsx`、`.xls` 或 `.csv`。
4. 配對欄位，例如玩家 A、牌組 A、玩家 B、牌組 B、勝方。
5. 查看 preview。
6. 按 `確認匯入`。

## 匯出 Excel

1. 到 `設定 → 匯入 / 匯出`。
2. 按 `匯出 Excel`。
3. 打開 Excel 後先看 `README` 和 `對局總表`。
4. 底線開頭的 sheets 是給 app 日後完整還原用。

## Cloud Backup 和群組共享

1. 到 `設定 → Cloud & Group`。
2. 輸入 email 並寄出登入連結。
3. 到 email 打開 magic link。
4. 回到網站後輸入群組碼。
5. 群組內成員可以上傳或下載同一份資料。

## 常見問題

### 為什麼找不到 Leader？

試試輸入 set code，例如 `OP01`、`OP-01`、`ST10`、`EB01`，或到 `設定 → Leader 管理` 加入本地別名。

### 匯入 OPCG Tracker 自己匯出的 Excel 會怎樣？

系統會自動辨認格式，顯示確認畫面。確認後才會用 Excel 裏的完整備份覆蓋本機資料。

### 群組碼是不是密碼？

是。群組碼等同邀請密鑰，只分享給可信任的人。
