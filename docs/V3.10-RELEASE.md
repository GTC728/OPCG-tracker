## 3.9.1 — Group & UX

### Mandatory group sync (join = sync, leave = stop)

- Removed the optional「即時協作」checkbox.
- `lastGroupCode` alone activates entity sync (`isGroupCollabActive`).
- Joining a group sets `groupCollabEnabled: true` and runs bootstrap automatically.
- Leaving a group clears membership and stops Realtime.

**Rationale:** Same friend circle should not fork into divergent databases because someone forgot to enable sync.

### Offline queue + pending UI

Reference pattern: foreground outbox queue (optimistic local write → durable queue → flush on `online` / visibility), similar to messaging apps and PWA offline-first guides.

Implementation:

| Piece | Location |
|-------|----------|
| IndexedDB queue (Dexie) | `src/lib/syncQueue.ts` |
| Enqueue on local persist | `notifyGroupCollabChange` in `src/lib/groupSync.ts` |
| Flush when online | `flushGroupCollabSync`, `useGroupCollab` `online` listener |
| Status banner | `src/components/layout/SyncStatusBanner.tsx` |

UI states:

- **Offline:** neutral warning bar —「離線模式 — 變更已儲存本機」
- **Pending:** brand bar —「{n} 項待同步」

Local recording always works; cloud push waits for connectivity.

### Player merge in group

- `mergePlayers` changes now sync via the same diff pipeline:
  - Remote **delete** for removed source player
  - **Upsert** target player (including aliases)
  - **Upsert** rewired matches / active matches

### Delete-only entity management (no archive)

- **Players:** removed archive/restore UI; only edit + delete (tombstone).
- **Leaders/decks:** removed archive step before delete; single delete action.
- Session archive unchanged (still useful for hiding old sessions without deleting history).

### Bottom navigation

- Emoji replaced with inline SVG icons (record / stats / history / settings).
- Height reduced: `min-h-11` → `min-h-9`, label `9px`.

### Match table deck names

- Removed `max-w-[8rem]` + hard `truncate` on deck labels.
- Uses `[overflow-wrap:anywhere]` so long leader names wrap instead of clipping.

Reference: tournament meta matrices (e.g. Limitless TCG dashboards, MTG meta analyzers) — larger grid + cell win% + match count + horizontal scroll.

- Matrix size **6 → 14** decks (by match volume in scope).
- Wider cells (`min-h-16`), larger min grid width.
- Row/column labels: `[word-break:break-word]`, full name in `title` tooltip.
- Still global **對位** tab only (profile heatmap remains backlog).
